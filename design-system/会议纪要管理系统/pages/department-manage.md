## Component Specs

### 部门树节点
- **Padding:** 8px 0
- **Indent:** 24px per level
- **Icon:** 部门 `Building`, 展开 `ChevronRight/Down`
- **Hover:** 背景 `#FAF5FF`

### 操作按钮
- **编辑:** `el-button` type="text", color `#7C3AED`
- **删除:** `el-button` type="text", color `#EF4444`
- **新增:** `el-button` type="primary", icon `Plus`

## Business Rules

| Rule | Description |
|------|-------------|
| 层级深度 | 最多5级 |
| 部门名称 | 同级唯一 |
| 删除限制 | 有用户关联不可删除 |

---

## Data Structure

```typescript
interface Department {
  id: string;
  name: string;
  parentId: string | null;
  description: string;
  sort: number;
  level: number;  // 1-5
  userCount: number;
  children?: Department[];
  createTime: string;
}
```

---

## Interaction Rules

| Action | Behavior |
|--------|----------|
| 点击树节点 | 选中并显示详情/操作 |
| 点击新增 | 打开新建部门对话框 |
| 点击编辑 | 打开编辑对话框，填充数据 |
| 点击删除 | 确认对话框，提示关联用户 |
| 拖拽节点 | 调整部门归属 (可选) |

---

## Accessibility

- [ ] 树节点支持键盘导航
- [ ] 删除操作有确认提示
- [ ] 表单验证有错误提示
- [ ] 层级深度有限制提示

---

## Anti-Patterns (禁止使用)

- ❌ 删除有用户的部门
- ❌ 层级超过5级无提示
- ❌ 部门名称重复无校验
- ❌ 树节点无法键盘操作
