## Data Structure

```typescript
interface OperationLog {
  id: string;
  userId: string;
  userName: string;
  operationType: string;
  operationDetail: string;
  ipAddress: string;
  userAgent: string;
  relatedType: 'minute' | 'template' | 'user' | 'task' | null;
  relatedId: string | null;

}
```

## Interaction Rules

| Action | Behavior |
|--------|----------|
| 筛选日志 | 重新加载列表数据 |
| 点击刷新 | 重新加载当前页数据 |
| 点击导出 | 导出当前筛选结果的Excel |
| 悬停详情 | 显示完整详情 tooltip |
| 切换分页 | 加载对应页数据 |

## Performance Requirements

| Metric | Requirement |
|--------|-------------|
| 日志查询 | < 500ms |
| 导出响应 | < 3秒 (可使用异步) |
| 列表加载 | < 1秒 |

## Responsive Behavior

| Breakpoint | Layout Change |
|------------|---------------|
| < 768px | 表格列精简，日期范围选择器全宽 |
| 768px - 1024px | 表格2/3列可见 |
| > 1024px | 完整5列表格 |

---

## Anti-Patterns (禁止使用)

- ❌ 日志无分页
- ❌ 详情过长无截断
- ❌ 导出无loading状态
- ❌ 敏感信息明文显示 (密码等)
