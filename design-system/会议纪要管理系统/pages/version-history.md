## Interaction Rules

| Action | Behavior |
|--------|----------|
| 点击查看 | 模态框显示该版本内容快照 |
| 点击对比 | 进入对比视图，选择要对比的版本 |
| 点击回滚 | 弹出确认对话框，显示回滚影响说明 |
| 选择对比版本 | 实时显示两个版本的差异对比 |
| 回滚确认 | 创建新版本，内容为历史版本，通知相关人员 |

---

## Diff Algorithm

- 使用 **diff-match-patch** 或类似算法
- 响应时间要求: < 3秒
- 高亮方式: 词级别对比，而非行级别

## Data Structure

```typescript
interface Version {
  id: string;
  minuteId: string;
  versionNumber: string;  // V1.0, V1.1, V2.0
  content: string;        // HTML快照
  modifierId: string;
  modifierName: string;
  modifyTime: string;
  remark: string;          // 版本备注
  isCurrent: boolean;
}

interface DiffResult {
  type: 'equal' | 'insert' | 'delete';
  text: string;
}
```

## Responsive Behavior

| Breakpoint | Layout Change |
|------------|---------------|
| < 768px | 版本对比变为单列显示，差异标注在行内 |
| 768px - 1024px | 版本对比2列，各列宽度50% |
| > 1024px | 版本对比2列，左列固定，右列可滚动 |

---

## Anti-Patterns (禁止使用)

- ❌ 回滚操作无确认对话框
- ❌ 对比结果不清晰
- ❌ 删除操作直接执行
- ❌ 历史版本可被物理删除 (仅逻辑删除)
