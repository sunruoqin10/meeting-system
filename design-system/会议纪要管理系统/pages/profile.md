## Data Structure

  id: string;
  name: string;
  email: string;
  phone: string;
  departmentId: string;
  departmentName: string;
  role: 'user' | 'admin' | 'dept_admin';

## Interaction Rules

| Action | Behavior |
|--------|----------|
| 点击更换头像 | 触发文件上传，选择图片后即时预览 |
| 点击编辑资料 | 表单变为可编辑状态 |
| 点击保存 | 调用更新API，成功后关闭编辑状态 |
| 点击修改密码 | 弹出密码修改对话框 |
| 密码修改成功 | 显示成功提示，重新登录 |

---

## Accessibility

- [ ] 所有表单有 label
- [ ] 编辑状态有视觉反馈
- [ ] 密码强度有文字提示
- [ ] 操作按钮有清晰的焦点状态

---

## Responsive Behavior

| Breakpoint | Layout Change |
|------------|---------------|
| < 768px | 头像和信息卡片垂直堆叠，居中对齐 |
| 768px - 1024px | 头像和信息卡片左右布局 |
| > 1024px | 完整两栏布局 |

---

## Anti-Patterns (禁止使用)

- ❌ 修改密码无强度验证
- ❌ 编辑取消无确认
- ❌ 头像上传无预览
- ❌ 表单无验证直接提交
