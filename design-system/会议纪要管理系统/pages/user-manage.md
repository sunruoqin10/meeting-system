## Component Specs

### 用户表格
- **Component:** Element Plus `el-table`
- **Border:** true
- **Row Height:** 56px
- **Avatar:** 36px 圆形

### 状态标签
- **Component:** Element Plus `el-tag`
- **正常:** type="success"
- **锁定:** type="warning"
- **禁用:** type="info"

### 操作下拉
- **Component:** Element Plus `el-dropdown`
- **Items:** 编辑 / 重置密码 / 锁定/解锁 / 删除

## Data Structure

```typescript
interface User {
  id: string;
  name: string;
  email: string;
  phone: string;
  departmentId: string;
  departmentName: string;
  role: 'user' | 'dept_admin' | 'admin';
  status: 'active' | 'locked' | 'disabled';
  avatar: string;
  lastLoginTime: string;
  createTime: string;
}
```

## Interaction Rules

| Action | Behavior |
|--------|----------|
| 点击新增 | 打开新建用户对话框 |
| 点击编辑 | 打开编辑对话框，填充数据 |
| 点击重置密码 | 确认后生成新密码，发送到用户邮箱 |
| 点击锁定/解锁 | 切换用户状态 |
| 点击删除 | 确认对话框，提示关联数据 |
| 筛选用户 | 重新加载列表数据 |

## Password Policy

| Rule | Requirement |
|------|-------------|
| 长度 | 至少8位 |
| 复杂度 | 大小写字母、数字、特殊字符 |
| 强制修改 | 90天 |

## Accessibility

- [ ] 表格支持键盘导航
- [ ] 操作按钮有 aria-label
- [ ] 删除操作有确认提示
- [ ] 表单验证有错误提示

---

## Anti-Patterns (禁止使用)

- ❌ 删除管理员账户
- ❌ 用户状态变更无提示
- ❌ 批量删除无确认
- ❌ 密码明文显示
